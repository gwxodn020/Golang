# Golang

## 환경 설정(Mac)

```
brew install go  //설치
export GOPATH=$HOME/go //Path 설정
export PATH=$PATH:$GOPATH/bin
```
---
## 프로젝트 설정

```
mkdir <프로젝트 이름>
touch <파일 이름>
go mod init <module-name>
go run main.go //파일 실행
```
